# PDF_copy_assis
自己在写论文时Ctrl C + Ctrl V  PDF 的内容出来格式各种问题，写了点代码来解决问题，顺便封装个GUI
当然，别计较bug一堆，代码丑陋，能用就行

# 功能
1. 中文复制整理
2. 截图翻译
3. 英文连词分割（正在开发中，version 1.0中没有）
